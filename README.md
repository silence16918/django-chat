# django-chat

### A small django chat room
<br/>

##### 1、简介: django+python3.6+mysql+pycharm
<br/>

##### 2、主要功能实现:
1,用户注册登录<br/>
2,ajax发送消息,长轮询接收消息<br/>
3,队列queue存储消息<br/>
<br/>
